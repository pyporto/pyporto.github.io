Using Python to learn how to code from scratch

Who are we?

We are the Python community of Porto. We are here to share our Python knowledge, sharpen our programming skills and also just to have fun. We believe that Python can be learned by anyone and because of its versatility, it is multi purpose.

PyPorto Academy

PyPorto Academy is a non-profit program for those who want to build there projects, but don’t have the technical knowledge. At PyPorto Academy, the students will be able to learn basic key concepts for writing code and for starting their own projects. The main goal of this program is to ramp-up anyone who has low or no programming skills into being able to start coding their own projects after the program.

Methodology

We are apologists of “learning by doing” and strive to make our program highly practical.

When presenting new concepts, we make sure that participants understand why they are learning them, grasp them well and apply them to their own specific problems.

Structure

The PyPorto Academy will run for one month every Saturday morning, between the 19th of January and 9th of February*.

19 Jan - Basic Concepts of Programming and Python; Creating working groups; Mentoring hours;
26 Jan - Basics of Data Analysis; Group work time; Mentoring hours;
02 Feb - Basics of Web Development; Group work time; Mentoring hours;
09 Feb - Finalizing the group work; Group Lunch; Prototype presentation; Feedback;

* Dates are tentative and a subject to change.
